# NudgeNotch Homebrew Tap

This repository contains the [Homebrew](https://brew.sh) cask for [NudgeNotch](https://github.com/suryanshkushwaha/nudge.notch) — gentle nudges for your wellbeing and focus, right in your MacBook notch.

## Installation

Install using Homebrew by running:

```
brew install --cask suryanshkushwaha/nudge-notch/nudge-notch --no-quarantine
```

## Updating

You can manually update through Homebrew by running:

```
brew upgrade --cask nudge-notch
```
